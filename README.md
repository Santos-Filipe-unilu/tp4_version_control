# REST API

This application allows us to create, read, update and delete Data from a Database

## Create
This allows us to create an entry in our database, this entry needs to have the values that match each field in order to be created and inserted into the database.

## READ
This operation allows us by giving a city name to retrieve the whole entry of that city name.

## Update
This operation allows us to update the population of a city by giving the name of the city that needs its population number changed and a new population number.

## Delete
The delete operation deletes a whole entry where the name given matches the respective city name.
